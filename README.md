## Modifications
### 1 - Bomb Kick
Bombs can now be kicked by the players.

### 2 - Dash
Players can now perform a quick dash. *(The bomb kick will scale with player's speed)*

### 3 - Build Wall
Players are now able to build bomb-destructable walls.
